# Codex Goal Diffusion Tasks

This directory contains prompt wrappers and Codex Goal workspaces for the
diffusion kernel tasks. Each task has its own folder, and both generated plans
and Codex optimization results should stay in that same folder:

```text
codex-goal-diffusion/<task>/
  prompt.md        # paste into a fresh Codex session to regenerate plan.md
  plan.md          # generated by the prompt
  baseline/         # Codex writes recovered baseline here
  solution/         # Codex writes candidate implementation here
  bench/            # Codex writes correctness/benchmark harness here
  docs/             # Codex writes run logs/results here
```

## Regenerate Plans

The task prompts now use the `brainstorming` skill from
`obra/superpowers/skills/brainstorming` for design exploration, then humanize
`gen-plan` to write the final Codex Goal plan.
All task prompts also point to
[`PLAN_GENERATION_SUPPLEMENT.md`](PLAN_GENERATION_SUPPLEMENT.md), which keeps
the comparison greenfield while requiring the public `diffusion/docs/` rules,
shape coverage, and shape-dispatch planning to be folded into generated plans.

Open a fresh Codex window for a task and paste:

```bash
cat codex-goal-diffusion/b200_diffusion_cutedsl_norm_scale_shift__multi_shape/prompt.md
```

The expected output is:

```text
codex-goal-diffusion/<task>/plan.md
```

## Launch One Task

After `plan.md` exists for a task, launch the actual optimization goal from
this directory:

```bash
scripts/launch_codex_goal_task.sh b200_diffusion_cutedsl_norm_scale_shift__multi_shape
```

Or use the numbered wrappers:

```bash
scripts/launch_tasks/k15_b200_diffusion_cutedsl_norm_scale_shift__multi_shape.sh
```

The launcher starts Codex in `--yolo` mode with this prompt:

```text
/goal follow the instruction in plan.md
```

It sets Codex's working directory to `codex-goal-diffusion/<task>/`, so the
kernel artifacts are produced under the Codex Goal task folder rather than
under `diffusion/kernels/`.

## Launch Many Windows

Print one command per task:

```bash
scripts/print_launch_commands.sh
```

Paste each printed command into a separate terminal window.

Useful environment overrides:

```bash
CODEX_GOAL_NO_CODEX=1 scripts/launch_tasks/k15_b200_diffusion_cutedsl_norm_scale_shift__multi_shape.sh
CODEX_MODEL=gpt-5.5 scripts/launch_tasks/k15_b200_diffusion_cutedsl_norm_scale_shift__multi_shape.sh
CODEX_GOAL_YOLO=0 scripts/launch_tasks/k15_b200_diffusion_cutedsl_norm_scale_shift__multi_shape.sh
```
